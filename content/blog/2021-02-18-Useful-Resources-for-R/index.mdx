# Cheat sheets
This [RStudio cheat sheets](https://rstudio.com/resources/cheatsheets/) website provided by RStudio contains cheat sheets for popular packages such as *ggplot2*, *readr*, and *Shiny*, as well as cheat sheets (e.g. [leaflet](https://github.com/rstudio/cheatsheets/raw/master/leaflet.pdf), [LaTeX](https://wch.github.io/latexsheet/)) contributed by R Users.
